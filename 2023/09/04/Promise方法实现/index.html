    Promise的含义

    所谓Promise，简单说就是一个容器，里面保存着某个未来才会结束的事件（通常是一个异步操作）的结果。从语法上说，Promise 是一个对象，从它可以获取异步操作的消息。Promise 提供统一的 API，各种异步操作都可以用同样的方法进行处理。

    promise对象有以下两个特点。

    1. 对象的状态不受外界影响，promise对象代表一个异步操作，有三种状态：pending (进行中)、fulfilled（已成功）和rejected（已失败）。
    2. 一旦状态改变，就不会再变，任何时候都可以得到这个结果。Promise对象的状态改变，只有两种可能：从pending变为fulfilled和从pending变为rejected。

    异步函数处理流程

    处理同步函数时then方法得到的状态为pending，此时需要保存回调函数，等待异步函数运行完成的到结果，在将结果给到resolve, reject之一的回调函数，then方法可能连续调用所以需要用数组保存回调函数

    class Promise {
    constructor(fun) {


        function resolve(data) {
            self.PromiseState = 'fulfilled'
            self.PromiseResult = data
            //如果是异步函数调用，只能在这里获取函数的返回，所以需要在这里调用之前存储的回调函数
            self.callbackFunctions.forEach(value => value.resolve())
        }

        function reject(data) {
            self.PromiseState = 'rejected'
            self.PromiseResult = data
            self.callbackFunctions.forEach(value => value.reject())

        }
        this.PromiseState = 'pending'
        this.PromiseResult = undefined
        const self = this
        fun(resolve, reject)
    }

    then(resolve, reject) {
        const self = this

        return new Promise((toresolve, toreject) => {
            function callback(type) {
                try {
                    const res = type(self.PromiseResult)
                    if (res instanceof Promise) {
                        res.then(r => toreject(r), e => toreject(e))
                    } else {
                        toresolve(res)
                    }
                } catch (e) {
                    toresolve(e)
                }
            }

            if (this.PromiseState === 'fulfilled') {
                callback(resolve)
            }

            if (this.PromiseState === 'rejected') {
                callback(reject)
            }
            // 异步函数调用
            if (this.PromiseState === 'pending') {
                this.callbackFunctions =[]
                this.callbackFunctions.push({
                    resolve:()=>{
                        callback(resolve)
                    },
                    reject:()=>{
                        callback(reject)
                    },
                })
            }

        })
    }
}
